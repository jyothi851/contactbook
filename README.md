### 📌 Project Overview

Built a **Command-Line Contact Book** in Python that allows users to efficiently manage their contacts through a simple and interactive menu.

### ✅ Features

* ➕ Add a new contact
* 📋 View all saved contacts
* 🔍 Search for a contact by name
* 🗑️ Delete an existing contact
* 💾 Store contacts using a **list of dictionaries**

### 🛠️ Technologies Used

* Python
* Lists
* Dictionaries
* Loops
* Conditional Statements
* Functions
* User Input Handling

This project helped me strengthen my understanding of **Python data structures, CRUD operations, menu-driven programming, and command-line application development**.
